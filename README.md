## Catatan Pembelajaran
1. Inheritance adalah mekanisme dalam OOP di mana sebuah class bisa mewarisi properties dan methods dari class lain.
2. Inheritance memungkinkan kita untuk reuse code dan membuat hubungan "is-a" antara classes.
3. Parent Class adalah class yang diwarisi. Child Class adalah class yang mewarisi.
4. super() harus menjadi statement pertama di constructor child class.
5. Method overriding adalah kemampuan subclass untuk menyediakan implementasi spesifik dari method yang sudah didefinisikan di parent class.
